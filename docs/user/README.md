# User Documentation

This folder contains documentation that is **shipped with the product** and intended for end users.

## Contents

- **user-guide.md** — Getting started, tutorials, and feature walkthroughs
- **configuration.md** — All CLI arguments, environment variables, and setup options
- **api-reference.md** — Programmatic API documentation
- **security.md** — Security practices, data privacy, and best practices
- **contributing.md** — Contribution guidelines for open-source users
- **assets/** — Banners, images, and visual media for documentation

## When distributing to users

Include everything in this folder in your product releases, documentation sites, and help systems. Users should have direct access to all content here.
