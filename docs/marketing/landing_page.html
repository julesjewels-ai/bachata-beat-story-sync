Features

Everything a dance videographer needs

Purpose-built for Latin dance content — not bolted onto a generic auto-editor.

Beat-Perfect Synchronisation

Cuts snap precisely to detected beats using librosa's beat tracker. Every edit lands exactly where the music demands it.

Motion-Intensity Matching

OpenCV scores each clip's kinetic energy. High-energy sections get your most dynamic footage. Quieter sections get slower, atmospheric clips.

Intelligent Clip Selection

Avoids repetition across a montage. Tracks which clips have been used and strategically reuses only the best moments when your library is limited.

Professional Visual Effects

FFmpeg xfade transitions (fade, wipe, slide), colour grades (vintage, warm, golden, B&W), and intro effects like bloom and vignette breathe.

B-Roll & Narrative Flow

Automatically schedules B-roll clips between high-intensity sequences. Creates a story arc — not just a wall of cuts — across the full montage.

Explainability & Planning

Dry-run mode generates a full Markdown segment plan before rendering. Export Excel analysis reports and structured JSON output for deeper inspection.

FAQ

Common questions

What video and audio formats are supported?
Audio input must be WAV format for reliable BPM analysis. Video clips can be any format that FFmpeg supports — MP4, MOV, MKV, AVI, and most other common containers. Output is always 720p H.264 MP4 for maximum compatibility with YouTube and social platforms.
Does my footage ever leave my machine?
No. The tool is local-first by design. All audio analysis, video scoring, and rendering happens on your own machine using your own CPU and GPU. Your footage never touches a cloud server. This is especially important for event videographers with client footage under NDA or consent agreements.
How long does processing take for a typical project?
For a typical 3-4 minute montage with 40-60 clips, analysis runs in under a minute on a modern machine. The FFmpeg render step depends on your hardware — expect 2-5 minutes on a mid-range laptop. YouTube Shorts batch export adds roughly 30 seconds per Short. The dry-run planning mode is near-instant.
Does it work for other dance styles — salsa, kizomba, reggaeton?
Yes. While the tool is optimised and named for Bachata, it ships with genre presets for salsa, kizomba, reggaeton, and more. Each preset adjusts pacing parameters, cut-point density, and B-roll scheduling to match the feel of that genre. You can also fully customise every parameter in montage_config.yaml.
What technical setup do I need to run it?
You need Python 3.13+, FFmpeg (available on macOS, Linux, and Windows), and the uv package manager. Run make install and the setup script handles everything else — Python virtual environment, librosa, OpenCV, MoviePy, and all dependencies. No Docker, no cloud account, no configuration wizard.
