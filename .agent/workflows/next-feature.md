# Next Feature Request Workflow

1. Read the skill instructions at `.agent/skills/feature-requests/SKILL.md`
2. Read the feature backlog at `.agent/skills/feature-requests/features.md`
3. Identify the highest-priority feature in `PROPOSED` or `PLANNED` state
4. Follow the skill workflow (Plan → Implement → Verify → Close)
5. Only work on **one feature at a time**
